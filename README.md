# Playwright-TheySaid

Simple Playwright + TypeScript framework scaffold.

## Installed dependencies

- `@playwright/test`
- `@types/node`

## Project structure

- `tests/` - test specs
- `fixtures/` - custom fixtures
- `helpers/` - helper utilities
- `pages/` - page objects

## Useful commands

- `npm test`
- `npm run test:ui`
- `npm run test:headed`
- `npm run test:debug`
